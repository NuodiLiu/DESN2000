# User Stories

The IDs below are stable references for the use-case mapping, dependency analysis, acceptance criteria, and Jira tasks. Scope and sprint names are kept separate from the IDs because priorities may change.

## MVP User Stories

**US-01. As a** student looking for shared housing, **I want to** see clear property details (rent, fees, location, real photos, reviews, nearby facilities), **so that** I can decide if a property meets my needs before contacting anyone.

**US-02. As a** student looking for housing, **I want to** see the verification status of listings, landlords, and housemates, **so that** I can avoid scams and make a safe choice.

**US-03. As a** student or long-term housemate, **I want to** create a lifestyle profile (cleanliness, sleep schedule, noise, guests, smoking, privacy, chores), **so that** other users can understand how I live.

**US-04. As a** student or long-term housemate, **I want to** compare housemate preferences (cleanliness, noise, smoking, guests, schedules, privacy, chores, bill payments), **so that** I can find a good match and reduce conflict.

**US-05. As an** international student, **I want** rental guidance in plain language and different languages, **so that** I can understand the Australian rental process.

**US-06. As a** student, **I want to** message landlords and potential housemates inside the app, **so that** I do not need to share my phone number or social media account too early.

**US-07. As a** student, **I want** guided questions and message prompts, **so that** I know what to ask before making a housing decision.

**US-08. As an** international student, **I want to** book a property inspection (video call, in-person visit), **so that** I can check the property before making a commitment.

**US-09. As a** future housemate, **I want to** create a pre-move-in agreement (chores, bills, guests, privacy, house rules), **so that** everyone understands the household expectations before moving in.

## Potential Extension User Stories

**US-10. As a** student, **I want to** save and compare listings and housemate profiles, **so that** I can review my options before making a decision.

**US-11. As a** student, **I want to** report suspicious users or listings, **so that** unsafe activity can be checked and other users can be protected.

**US-12. As a** shared-house resident, **I want to** assign, rotate, and track chores, **so that** household work is shared fairly.

**US-13. As a** shared-house resident, **I want to** split shared costs by category and track payments (amount, payer, due date, payment status), **so that** everyone knows what they owe and pays on time.

**US-14. As a** resident, **I want to** raise sensitive household issues without showing my name, **so that** I can discuss problems without creating more conflict.

## Property Application and Management User Stories

**US-15. As a** property manager, **I want to** publish complete property listings (photos, rent, fees, house rules, availability), **so that** I can find suitable tenants and reduce vacancy time.

**US-16. As a** student applying for a property, **I want to** submit my profile and rental details, **so that** the property manager can review my application.

**US-17. As a** property manager, **I want to** review complete rental applications (identity, contact details, preferences, rental history), **so that** I can screen applicants and choose reliable tenants.

**US-18. As a** property manager, **I want to** manage property enquiries in one dashboard, **so that** I can keep communication organised and reduce repeated work.

**US-19. As a** property manager, **I want to** send announcements to current tenants, **so that** everyone receives the same property information.

**US-20. As a** property manager, **I want to** match applicants with properties (budget, location, room needs, lease period), **so that** I can find a suitable tenant for each property.

**US-21. As a** property manager, **I want to** send rent reminders and track payment status, **so that** I can identify and follow up on overdue payments.
