# Part 1 Asset Index

This folder is the source asset library for Part 1 and the reference set for Part 2.

## Main Categories

- `brand/`: UNSW logo and shared branding assets.
- `synthesis/`: mind maps, mood boards, and synthesis artefacts.
- `survey/charts/`: survey chart images used directly in the Week 1-4 group report.
- `survey/reports/`: source survey report PDFs.
- `personas/`: persona and journey-map images used in the reports.
- `personas/notes/`: persona source notes.
- `planning/`: user flow, Gantt chart, and architecture diagrams.
- `research_notes/`: research background, interview notes, and problem statement notes.
- `user_stories/`: early user story notes.
- `media/`: interview or research media files.
- `archive/duplicate_exports/`: duplicate exports kept for traceability.
- `archive/unused_group_figures/`: older group-report figure exports that are not referenced by the current `.tex` files.

## LaTeX Entry Points

- `../design_journal_week4_housemate_draft.tex`
- `../week1to4_group_report/group_report_week4.tex`

Both files now load images from this `assets/` folder through `\graphicspath`.
