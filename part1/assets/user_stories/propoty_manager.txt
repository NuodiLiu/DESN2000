

User Story
As a property manager, I want to advertise available rental properties to students efficiently, so that I can reduce vacancy periods and find suitable tenants faster.

As a property manager, I want to receive complete and accurate tenant information before arranging inspections, so that I can save time during the screening process.

As a property manager, I want to communicate with prospective tenants through a centralised platform, so that I can manage enquiries more efficiently.

As a property manager, I want to verify tenant profiles and rental histories, so that I can minimise risks and select reliable tenants.

As a property manager, I want to match properties with suitable tenants based on their preferences and requirements, so that tenancy issues can be reduced.

Acceptance Criteria

● Property listings can be uploaded within 5 minutes.
● Photos and rental details can be attached.
● Students can contact the property manager directly through the platform.
● The property manager can view all enquiries in one dashboard.

Background
Sarah works for a property management company that manages student accommodation and private rental properties around Sydney universities. She handles tenant enquiries, property inspections, lease agreements, and maintenance requests.

Due to the increasing demand for student accommodation, Sarah receives a large number of applications from both domestic and international students every semester. Student housing demand remains high while accommodation supply is limited.
